Getting API Keys

Recommended:

Not all features all are access or scrapped without API keys, So recommened to signup following websites and get your keys to access OSINTsploit with Complete Features in it

1.HaveIbeenPwned
  • Visit https://haveibeenpwned.com/API/Key
  • Enter your valid Mail ID
  • Api Documentation https://haveibeenpwned.com/API/v3
  • Place your API in Settings Tab
2.Hunter.io
  • Visit https://hunter.io/users/sign_up?utm_medium=api
  • Create a new account or sign up with google and continue
  • Api Documentation https://hunter.io/api-documentation/v2
  • Visit this link https://hunter.io/api-keys for getting your API key
  • Place your API in Settings Tab
3.HLR Lookup
  • Visit https://www.hlr-lookups.com/
  • Create a new account and complete the process
  • Api Documentation https://www.hlr-lookups.com/en/api-docs
  • Visit your profile and get API Key
  • Place your API in Settings Tab
4.IP Stack
  • Visit https://ipstack.com/product
  • Click get free api key
  • Create a new account and complete the process
  • Api Documentation https://ipstack.com/documentation
  • Visit your profile and get API Key
  • Place your API in Settings Tab
4.Virus Total
  • Visit https://developers.virustotal.com/reference
  • Create a new account by clicking the link here https://www.virustotal.com/#/join-usand complete the process
  • Visit your profile and get API Key
  • Place your API in Settings Tab
5.Shodan Key
  • Visit https://developer.shodan.io/api/requirements
  • Create a new account and complete the process
  • Read the API Documentationhttps://developer.shodan.io/api
  • Visit your profile and get API Key
  • Place your API in Settings Tab
6.Google Maps
  • Go to the Google Cloud Platform Console.
  • Click the project drop-down and select or create the project for which you want to add an API key.
  • Click the menu button and select APIs & Services > Credentials.
  • On the Credentials page, click Create credentials > API key.
    The API key created dialog displays your newly created API key.
  • Click Close.
    The new API key is listed on the Credentials page under API keys.
    (Remember to restrict the API key before using it in production.)
