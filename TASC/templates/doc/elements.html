Features & Elements

Phone Number

Syntax : phone:number

Phone number module grabs the user's information about :

  • Subscribe Status
  • State
  • Country
  • IMSI
  • MCC MNC
  • Ported
  • Original Network
  • Ported Network
  • Roaming
  • Roaming Country

Torrent Download Tracker

Syntax : ip:address

  • Title
  • Category
  • Size
  • Date

IP Analysis

Syntax : ip:address

  • City
  • State
  • Country
  • Proxy / VPN / TOR relay
  • Proxy Type
  • ISP
  • Domain
  • Usage Type
  • ASN
  • Comain Name
  • Latitude
  • Longitude

Username Enumeration


Syntax : social:username

  • Name
  • Username
  • Work
  • Education
  • Current Location & Home Town
  • Tweets and Posts
  • Profile Picture
  • Url
  • Posts Location
  • Following & Followers Count

Email Analysis

Syntax : email:email_address

  • Domain
  • Url
  • Extracted Date
  • Last seen
  • Still on Page or not

Web OSINT


Syntax : email:email_address
